# 📦 StockFlow

Модульная система управления складским инвентарём с динамическими типами объектов.

**FastAPI · React · PostgreSQL**

---

## Что это

Вместо жёстких категорий — вы сами описываете, что хранить и какие у этого атрибуты.
Создали тип "Футболка" с полями `бренд`, `размер`, `состояние`, `цена` — система сама строит формы, таблицы, фильтры и правила уведомлений.

## Ключевые фичи

- 🧩 **Конструктор типов** — любой объект с любыми полями без изменения кода
- 🗺️ **Иерархия хранения** — Склад → Помещение → Стеллаж → Полка → Ячейка
- 🏷️ **Автоматический SKU** — генерируется на основе типа и места хранения
- 🔔 **Умные уведомления** — товар лежит 60 дней? Система предложит уценку
- 📊 **Дашборд** — оборачиваемость, залёжавшиеся позиции, заполненность склада
